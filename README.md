"# AssignmentImageRotateRest" 

This project is created for Rotating image by taking input image url from user.

For calling API, you can use POSTMAN client

API URL :

http://ec2-54-214-110-77.us-west-2.compute.amazonaws.com:8080/api/rotateImage

Headers :

ContentType : application/x-www-form-urlencoded

Parameters :

imageURL : http://rku.ac.in/wp-content/uploads/2018/02/image-1.png


Then same Image will be returned with applied rotation of 90 clock-wise.
